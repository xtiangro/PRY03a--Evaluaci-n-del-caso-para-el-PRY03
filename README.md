# PRY03a--Evaluaci-n-del-caso-para-el-PRY03
Instrucciones
Con base a la configuración del campus del ITSX considerar según se hayan distribuidos los retos, la necesidad de optimizar una ruta desde un punto origen a un punto destino (edificios), en función de:

EQUIPO TIERRA: Generar una ruta donde se minimice la distancia en metros a recorrer caminando y se minimice el número de escalones que se deban subir o bajar.

EQUIPO AIRE-FUEGO: Generar una ruta donde se minimice la distancia en metros a recorrer caminado y también se minimice el recorrido que debe hacer por andadores con árboles cuya fronda pueda representar un riesgo por caída de un rayo , desgajamiento o caída del árbol.

EQUIPO AGUA: Generar un ruta donde se minimice la distancia en metros a recorrer caminando, pero que represente mayor oportunidad de refugio o zonas techadas.

Para esta sesión el equipo debe:
Analizar la matriz que generaron de manera individual para representar la configuración del campus y acordar una sola representación (la más adecuada), considerando el objetivo perseguido y las consideraciones del campus.
Analizar, definir y agregar, los datos faltantes para resolver según sea necesario para cada caso.
Acordar en grupo, el uso de un framework de programación, ya que al final todo será integrado de tal manera que cualquier persona indique un origen y un destino, y en función de las condiciones meteorológicas se utilice el método diferenciado para darle una sugerencia.
ENTREGABLES DEL DÍA DE HOY:

Estructura de la matriz.
Datos cargados en la matriz.
Marco de desarrollo instalado y acordado por el grupo.
Primeros avances de programación.
